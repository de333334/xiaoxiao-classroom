## 三、第一件：万能缓冲句（你的"缓兵之计"）

接不上、不想答、没听清、被夸——先一句话把球接住或挡回去，别硬着头皮胡说。

分**四类**，存进你的"金句本"：

| 情况 | 万能缓冲句（直接抄） |
|------|------|
| 接不住话 | "这个角度我还真没想过，你接着说。" |
| 接不住话 | "我整理下思路，待会细说。" |
| 不想答（隐私/不想聊） | "这个我不太方便聊，咱换个话题？" |
| 不想答 | "你先说你的，我听听。" |
| 没听清 | "你刚说的那点，能再讲一遍吗？" |
| 被夸 | "哈哈被你一夸我都不好意思了，其实还差得远。" |
| 被夸 | "谢谢，你这么一说我更有动力了。" |

**口诀**：接不住你接着说，不想答巧转话题，没听清请人再说，被夸了稳稳接住。

> 小明踩过的坑：以前被夸只会傻笑或猛摆手"没有没有"，反而尴尬。用缓冲句把夸奖接住，又不显得飘。

## 四、第二件：话别说满（替换表）

把"绝对、肯定、必须"换成"可能、通常、也许"，留余地。

| 原来（太满） | 换成（留余地） |
|------|------|
| 绝对没问题 | 通常没问题 |
| 我肯定能搞定 | 我尽量 |
| 必须听我的 | 建议这样，也听听你的 |
| 一定办成 | 基本能成，有进展第一时间告诉你 |

**注意**：有把握可以说"基本能成""我有九成把握"，别用"绝对"。笃定让人信赖，打包票做不到就砸锅。

**口诀**：说七分留三分，话到嘴边缓一缓。

## 五、第三件：场景预设填空卡

提前给高频场面写好万能回应，现场直接"掏卡"。

| 高频场景 | 预设回应（填空式） |
|------|------|
| 被突然点名发言 | "我整理下思路，待会细说。" |
| 被当众问倒 | "我先记下来，回头好好想，再回你。" |
| 求人帮忙 | "这事有点难，不过我试试，有消息第一时间告诉你。" |
| 被问隐私 | "这个我不太方便聊，不过你最近怎么样？" |

**怎么挑场景？** 记住三个字：**常、怕、卡**
- 常——你经常遇到的；
- 怕——你最怕碰到的；
- 卡——上次卡住过的。

把这三类各写一句，存进手机备忘录，就是你的"预设卡"。

**口诀**：常怕卡，先写卡，遇事掏卡不抓瞎。

## 六、第四件：榜样观察册

你身边那个"说话让人舒服"的人，就是现成教材。观察三件事，抄下他的原话：

1. **怎么接话**——是打断，还是先点头"嗯，我懂你的意思"？
2. **怎么拒绝**——是冷脸，还是先谢后拒"谢谢邀，这回真分身乏术"？
3. **怎么提意见**——是对人（"你这人真轴"），还是对事（"这个环节如果换个顺序会不会更好"）？

## 七、第五件：角色扮演练习本（练到不慌）

找朋友，或对着镜子，模拟尴尬场面，把缓冲句和预设卡练熟。

**四步练法**（每天只练一类场景，15分钟就够）：
1. **热身**：先聊个轻松的，别一上来就压力拉满。
2. **上场**：朋友演"刁难者"，你接招；没人陪就**对镜练**、用 AI 模拟。
3. **复盘**：哪里没接好？换成哪句会更顺？
4. **鼓励**：不批评，只夸一点点。

**关键**：预演最坏情况——对方可能怎么怼我、怎么追问，我一句句接住。练熟了，真到现场就像背过的课文，不慌。

**口诀**：找人演，对镜练，最坏情况先预演。

## 八、第六件：向书学、向故事学（没榜样也别慌）

身边没有榜样？书里、故事里到处是。看别人踩的坑，就是替你先试错一次。

- 《非暴力沟通》：说**事实**不说**评判**（"你迟到三次" vs "你总不守时"）。
- 《人性的弱点》：让对方多说，别急着证明自己对。
- 访谈、综艺里的救场名场面：看一遍，下次轮到你不慌。

**口诀**：没榜样，看书闯，别人踩坑我成长。

> 例子：演员黄渤在金马奖被主持人调侃穿得像睡衣、还被呛"金马是我家不是你家"。他没急没躲，回了一句"我只见过人骑马，头回见马骑人"，把尴尬接住又化解，全场鼓掌。

## 九、第七件：三问自查镜（开口前照一照）

开口前，心里默念三问，再数一二三：

1. **该说吗？** —— 是必要的，还是我一时上头？
2. **负责吗？** —— 我能为这句话负责吗？
3. **能补吗？** —— 说错了能圆回来吗？

三问过一遍，话就稳了。

## 十二、随堂自测（写下来，再对答案）

**题1**：同学突然在班会上点你名发言，你完全没准备，哪句最得体？
- A. "我没想法。"
- B. "我整理下思路，待会细说。"
- C. "这问题太蠢了。"

**题2**：把"我绝对能帮你搞定"换成哪句更稳？
- A. "我一定办成，放心。"
- B. "我尽量，有进展第一时间告诉你。"
- C. "这事包在我身上。"

**应用题**：亲戚当众问"你一个月挣多少"，你不想答。请用"万能缓冲句 + 场景预设"写一句你会怎么说。

---

## 十三、参考答案

- **题1 → B**。"我整理下思路，待会细说"用预设卡争取时间，不硬撑也不怂；A 太丧、C 得罪人。
- **题2 → B**。"尽量"把"绝对"换成留余地，做不到也不砸锅；A、C 都是打包票。
- **应用题参考**："这个我不太方便聊，不过您最近身体怎么样？"——缓冲句挡回隐私，再把话题抛回对方，不伤和气。
- **费曼复述要点**：万能缓冲句分四类（接不住／不想答／没听清／被夸）；话别说满就是把"绝对、肯定、必须"换成"可能、通常、也许"。讲不明白，就是还没真懂，多讲两遍就通了。
